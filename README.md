# tg_emailer

## Описание
`tg_emailer` — это бот для Telegram, предназначенный для удобной отправки электронных писем непосредственно из мессенджера. Этот инструмент позволяет пользователям быстро и эффективно управлять своей почтой через простой и интуитивный интерфейс Telegram.

## Особенности
- Отправка писем через Telegram-команды.
- Работа с несколькими учетными записями электронной почты.
- Настройка шаблонов писем для автоматизации процессов.
- Полностью настраиваемый и расширяемый дизайн.

## Запуск с Docker Compose
Чтобы запустить бота с использованием Docker Compose, выполните следующие шаги:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/h4rd1/tg_emailer.git
   cd tg_emailer
   ```

2. Создайте файл `.env` и заполните необходимые переменные окружения:
   ```bash
   TELEGRAM_BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
   EMAIL_PROVIDER=YOUR_EMAIL_PROVIDER
   EMAIL_USERNAME=YOUR_EMAIL_USERNAME
   EMAIL_PASSWORD=YOUR_EMAIL_PASSWORD
   ```

3. Запустите контейнеры с помощью Docker Compose:
   ```bash
   docker-compose up -d
   ```

Теперь ваш бот готов к работе!

## Команда запуска
Используйте следующую команду для начала взаимодействия с ботом:

```bash
docker exec -it tg_emailer_bot /bin/bash
```

Это позволит вам войти в контейнер и вручную проверить состояние бота.
