# tg_emailer

Простой инструмент для отправки электронных писем через Telegram бот.

## Описание

Данный проект позволяет отправлять электронные письма непосредственно из Telegram. Это удобно для тех, кто хочет автоматизировать отправку писем или настроить массовые рассылки.

## Установка

Чтобы установить и запустить проект, выполните следующие шаги:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/h4rd1/tg_emailer.git
   ```

2. Создайте нового Telegram бота через @BotFather и получите токен.

3. Внесите изменнения в **.env** :
   ```bash
   nano .env
   ```

4. Запустите приложение:
   ```bash
   docker build -f dockerfile -t tg-emailer .
   docker-compose up 
   ```

## Использование

После запуска приложения вы сможете взаимодействовать с ним через Telegram. Бот предложит вам выбрать команду для отправки письма. Просто введите адрес электронной почты, тему и тело письма, и оно будет отправлено автоматически.

## Возможности

- Отправка индивидуальных писем.
- Массовая рассылка.
- Шаблоны писем.
- Интеграция с различными почтовыми сервисами.

## Поддерживаемые сервисы

На данный момент поддерживаются следующие почтовые сервисы:
- SMTP-серверы (например, Gmail, Yandex).
